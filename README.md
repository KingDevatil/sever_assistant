# 服务器后台指令快捷输入工具

一个基于PyQt5开发的服务器管理工具，支持多服务器管理、指令管理、文件上传下载等功能。

## 功能特点

- **多服务器管理**：支持添加、编辑、删除、复制服务器
- **指令管理**：支持分类管理指令，可添加带参数的指令
- **文件传输**：支持文件上传下载
- **实时输出**：支持持续命令（如tail -f）的实时输出
- **深色模式**：支持浅色/深色主题切换
- **布局设置**：可自定义界面布局参数
- **配置导出**：支持配置的导入导出

## 技术栈

- Python 3.7+
- PyQt5
- paramiko

## 安装

### 直接运行

1. 确保安装了Python 3.7+
2. 安装依赖：
   ```bash
   pip install PyQt5 paramiko
   ```
3. 运行：
   ```bash
   python main.py
   ```

### 可执行文件

在`dist`目录中已提供打包好的可执行文件，直接运行即可。

## 使用说明

### 添加服务器
1. 在左侧"服务器列表"标签页点击"添加服务器"
2. 填写服务器信息（名称、地址、端口、用户名、密码）
3. 点击"保存"

### 连接服务器
1. 右键点击服务器，选择"连接"
2. 连接成功后会在右侧出现服务器标签页

### 执行指令
1. 在指令管理中添加指令
2. 连接服务器后，在服务器标签页中点击指令按钮执行

### 文件上传
1. 点击"上传文件"按钮
2. 选择本地文件
3. 文件会上传到服务器当前目录

### 文件下载
1. 点击"下载文件"按钮
2. 输入要下载的文件路径
3. 选择保存位置

## 配置文件

- `servers.json`：服务器配置
- `commands.json`：指令配置
- `settings.json`：程序设置

## 打包方法

使用PyInstaller打包：

```bash
pyinstaller --noconfirm "服务器后台指令快捷输入工具.spec"
```

## 注意事项

- 密码以明文存储在配置文件中，请确保配置文件安全
- 支持的持续命令：tailf, tail -f, top, watch
- 执行持续命令时会显示"停止命令"按钮
